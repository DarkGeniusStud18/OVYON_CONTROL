# OVYON CONTROL - GENIUS MODE DEMONSTRATION GUIDE

Ce document explique comment orchestrer la démonstration la plus impressionnante possible pour votre soutenance.

## 🧠 1. Le Cœur du "Broken Genius" : Aion Brain
Vous avez maintenant deux scripts IA :
1.  `aion.py` : L'interface réactive (attend vos ordres).
2.  `aion_brain.py` : Le cerveau proactif (analyse les erreurs et suggère des actions).

**Pour choquer le jury :** Laissez `aion_brain.py` tourner en arrière-plan. Si vous laissez la porte ouverte trop longtemps pendant votre présentation, Aion enverra une notification "Sécurité" sur le Dashboard.

## 🎭 4. Déroulement Master-Class (Plan de Soutenance)

### Phase 1 : L'immersion (3 min)
*   Montrez le **Plan 2D** sur une tablette. Touchez une pièce, la lumière s'allume avec un effet PWM (fondu).
*   *L'astuce :* Dites au jury que le plan respecte l'architecture locale.

### Phase 2 : Le Choc Multilingue (5 min)
*   Ne parlez pas qu'en Français. 
*   *"Aion, allume le salon"* -> *"Ma lampu"* -> *"Tan ina"*.
*   Expliquez que vous utilisez **Gemini AI** pour comprendre l'intention derrière les mots, même complexes.

### Phase 3 : L'Intelligence Proactive (5 min)
*   Allez sur l'onglet **"Auto"** (Automatisations).
*   Approchez un briquet (prudemment) du capteur DHT11. 
*   La température monte -> Le ventilateur (Prise 1) s'enclenche seul.
*   Dites : *"Le système ne m'écoute pas seulement, il comprend mon environnement."*

### Phase 4 : La Vision Futuriste
*   Ouvrez l'onglet **"Vision"**. Présentez OVYON non comme un projet d'école, mais comme une startup pour l'Afrique.
